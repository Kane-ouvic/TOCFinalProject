# TOC Final Project

## 1. How to use

* start from local
```
pip install -r requurments
ngrok http 8000
uvicorn main:app --reload
```

## 2. 功能

### 生活小幫手
* 聊天 (OpenAI chatGPT)
* 隨機決定想買的選項
* 代辦事項清單 (資料庫: Microsoft Azure Table)
* 號碼對獎 (資料庫: Microsoft Azure Table)
* 尋找附近餐廳 (資料來源: GCP map)
* 天氣預報 (資料來源: 中央氣象局)
